# NextStep_back

## NextStep API

NextStep API est le backend de l’application NextStep. Il s’agit d’une API REST développée avec Express.js et connectée à Supabase pour la gestion des données.

Elle permet de gérer les candidatures à travers des opérations CRUD (création, lecture, mise à jour, suppression), tout en centralisant la logique métier et en assurant la communication entre le frontend (Vue.js) et la base de données.

## Fonctionnalités

* Création d’une candidature
* Récupération des candidatures
* Mise à jour des informations et du statut
* Suppression d’une candidature

## Technologies

* Node.js
* Express.js
* Supabase
